## مقدمة عن المنصة

### ما هي منصة رحماء بينهم؟

منصة رحماء بينهم الرقمية المتكاملة هي نظام مؤسسي شامل يدمج بين:
- **الواجهة الرقمية العامة (Website/Frontend)**: بوابة المجتمع والمانحين
- **نظام إدارة المحتوى (Sanity CMS)**: إدارة المحتوى التحريري والمرئي
- **لوحة التحكم الإدارية (Admin Dashboard)**: إدارة العمليات والتفاعلات
- **قاعدة البيانات (PostgreSQL/Neon)**: تخزين البيانات التفاعلية

### الرؤية
أن تصبح المنصة النظام الرقمي المرجعي الأول للعمل الإنساني والتنموي في اليمن، عبر تكامل غني بين واجهة المجتمع وأنظمة الإدارة.

### الرسالة
تقديم تجربة رقمية موحدة وموثوقة تعكس رسالة المؤسسة وتبرز أثرها، عبر ربط لحظي بين بيانات العمل الميداني والإداري.

## لوحة التحكم الإدارية المتكاملة

### المكونات المتاحة

#### 1. AdminStats (بطاقات الإحصائيات المتكاملة)
- **4 بطاقات إحصائية** من قاعدة البيانات مباشرة:
  - 📧 **الرسائل**: العدد + الجديد
  - 💖 **التبرعات**: العدد + المبلغ الإجمالي (ريال)
  - 👥 **المتطوعون**: العدد
  - 📈 **أداء الموقع**: 95%
- **تحديث تلقائي** من PostgreSQL
- **زر تحديث** يدوي
- **معالجة أخطاء** مع زر إعادة المحاولة

#### 2. VideoManager (مدير الفيديو المتكامل)
- **4 بطاقات إحصائية** (إجمالي، منشور، مميز، قصة)
- **قائمة فيديوهات** قابلة للتوسيع
- **أزرار تحكم**: تعيين/إلغاء التميز، نشر/إلغاء نشر
- **رابط مباشر** إلى Sanity Studio
- **3 حالات واجهة**: تحميل، خطأ، فارغ

#### 3. Hero Video Controls (تحكم فيديو الهيرو)
- زر تشغيل/إيقاف الفيديو
- زر كتم/تفعيل الصوت
- عرض حالة المصدر (Sanity / رابط خارجي / محلي)
- عرض حالة الصورة البديلة (Poster)

## نظام إدارة الفيديو المتكامل

### المكونات

#### 1. IntroVideoPlayer (مشغل الفيديو التعريفي)
- **الملف الافتراضي**: `/videos/hero-background.mp4`
- **معالجة الأخطاء (3 حالات)**:
  - 🌀 **تحميل**: سبينر دوار + نص "جاري تحميل الفيديو..."
  - 📡 **لا يوجد اتصال**: أيقونة تحذير + رسالة + زر إعادة محاولة
  - ❌ **فشل التحميل**: أيقونة خطأ + رسالة متغيرة + زر إعادة + صورة poster بديلة
- **مراقبة الاتصال**: أحداث `online`/`offline` عبر `navigator.onLine`
- **عناصر التحكم**: تشغيل/إيقاف، كتم الصوت، شريط التقدم، سرعة التشغيل، ملء الشاشة، تكبير/تصغير

#### 2. VideoShowcase (عرض الفيديوهات)
- جلب الفيديوهات من Sanity عبر GROQ
- بطاقات فيديو مع صور مصغرة
- مشغل فيدييو احترافي (ProfessionalVideoPlayer)
- زر قناة اليوتيوب
- تصنيف وعرض حسب الفئة

#### 3. VideoManager (إدارة الفيديو - Dashboard)
- إحصائيات حية من Sanity
- تحكم كامل (نشر، تميز، تعديل)
- تكامل مباشر مع Sanity Studio

### تدفق عمل الفيديو

```
1. Sanity CMS (video type)
   ↓
2. VideoShowcase (public) / VideoManager (admin)
   ↓
3. IntroVideoPlayer (hero-background.mp4)
   ↓
4. Error Handling (3 حالات)
   ↓
5. Fallback (poster image)
```

## التكامل الكامل مع قاعدة البيانات

### AdminStats - الاتصال المباشر بـ PostgreSQL

#### الاستعلامات المستخدمة:

1. **الرسائل (`messagesQueries.getStats`)**:
   ```sql
   SELECT 
     COUNT(*) as total,
     COUNT(*) FILTER (WHERE status = 'new') as new,
     COUNT(*) FILTER (WHERE status = 'read') as read,
     COUNT(*) FILTER (WHERE status = 'replied') as replied
   FROM messages
   ```

2. **التبرعات**:
   ```sql
   SELECT COUNT(*) as total, COALESCE(SUM(amount), 0) as amount FROM donations
   ```

3. **المتطوعون**:
   ```sql
   SELECT COUNT(*) as total FROM volunteers
   ```

#### الميزات:
- ✅ **تحديث مباشر** من قاعدة البيانات
- ✅ **حساب تلقائي** للمبالغ والإحصائيات
- ✅ **معالجة أخطاء** مع رسائل واضحة
- ✅ **زر إعادة المحاولة** في حالة الفشل

### الصفحات المتكاملة مع قاعدة البيانات

#### 1. MessagesPage
- **عرض** جميع الرسائل مع فلاتر (الكل، جديد، مقروء، تم الرد، مؤرشف)
- **تحديث الحالة**: جديد → مقروء → تم الرد → مؤرشف
- **حذف** الرسائل مع تأكيد
- **بريد إلكتروني** ورقم هاتف وموضوع
- **تاريخ** ووقت الرسالة
- **عدد غير مقروء** مع تمييز بصري

#### 2. DonationsPage
- **إحصائيات علوية**: إجمالي التبرعات، المبلغ الإجمالي، معلق، مكتمل
- **فلاتر**: الكل، معلق، مكتمل، فاشل، مسترجع
- **تأكيد التبرعات**: تحويل من معلق إلى مكتمل
- **عرض** المتبرع (مجهول/اسم)، البريد، الهاتف، المبلغ، العملة
- **المشروع** المرتبط (إن وجد)
- **ملاحظات** التبرع
- **طريقة الدفع** ونوع التبرع

#### 3. VolunteersPage
- **إحصائيات علوية**: إجمالي المتطوعين، نشط، قيد المراجعة، ساعات التطوع
- **فلاتر**: الكل، نشط، غير نشط، قيد المراجعة
- **قبول/رفض** المتطوعين
- **عرض** الاسم، البريد، الهاتف، المجال (إغاثة، تعليم، صحة، إدارة، تسويق)
- **ساعات التطوع** المحققة
- **الدافع** للتطوع
- **تاريخ التسجيل**
