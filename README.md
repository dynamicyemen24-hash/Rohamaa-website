# مؤسسة رحماء بينهم
## منصة رقمية متكاملة لإدارة العمل الإنساني والخيري

---

## 📋 نظرة عامة

منصة رقمية حديثة ومتطورة تم تطويرها خصيصاً لمؤسسة رحماء بينهم، تهدف إلى تسهيل إدارة البرامج الإنسانية والتنموية، وتمكين الفريق من إدارة المحتوى والبيانات بكفاءة عالية مع أعلى معايير الأمان والأداء.

## 🎯 ما تقدمه المنصة

### للزوار والراغبين في المساهمة:
- 📱 **صفحة رئيسية** تعرض رسالة المؤسسة وأهدافها
- 📰 **أخبار وفعاليات** محدثة باستمرار
- 🤝 **قصص نجاح** ملهمة من المستفيدين
- 💰 **التبرع الإلكتروني** بسهولة وأمان
- 🙋 **التسجيل للتطوع** في المشاريع المختلفة
- 📞 **صفحة اتصال** للاستفسارات

### لفريق العمل:
- 📊 **لوحة تحكم إدارية** متكاملة
- ✏️ **إدارة الأخبار** والإعلانات
- 📁 **إدارة المشاريع** والميزانيات
- 📈 **تقارير وإحصائيات** مفصلة
- 👥 **إدارة المتطوعين** والمتبرعين
- 🔐 **صلاحيات متدرجة** للمستخدمين

## 🎨 مميزات المنصة

### تصميم عصري ومتجاوب
- تصميم احترافي يعكس هوية المؤسسة
- متجاوب مع جميع الأجهزة (كمبيوتر، آيباد، موبايل)
- واجهة عربية سهلة الاستخدام
- دعم كامل للغة العربية واتجاه RTL

### أمان عالي
- حماية متقدمة للبيانات والتشفير
- صلاحيات وصول مختلفة للموظفين
- نظام مصادقة آمن مع JWT + refresh token
- نسخ احتياطية دورية
- تنشيط Service Worker للتخزين المؤقت الذكي
- تشغيل بدون اتصال (PWA)

### أداء موثوق
- سرعة عالية في التحميل (< 3 ثوانٍ)
- عمل مستقر على مدار الساعة
- تخزين مؤقت ذكي مع Service Worker
- دعم العمل بدون اتصال
- lazy loading للصفحات والأصول
- Code splitting محسّن

## 🏗️ البنية التقنية

### الفريق التقني المسؤول
**شركة ديناميك لخدمات البرمجيات**

### البيانات التقنية
| البند | التفاصيل |
|-------|---------|
| لغة البرمجة | React 18 + TypeScript 5 |
| التصميم | Tailwind CSS 4 + MUI 7 |
| إدارة الحالة | React Context + CQRS |
| قاعدة البيانات | Supabase / PostgreSQL |
| CMS | Sanity.io (Headless CMS) |
| الاستضافة | Vercel (سحابية عالية الموثوقية) |
| الشهادة الأمنية | HTTPS / SSL تلقائي |
| Service Worker | متقدم مع استراتيجيات تخزين مؤقت |
| Authentication | JWT مع refresh token |
| CI/CD | GitHub Actions |
| الحزم | pnpm (workspace monorepo) |

## 🚀 البدء السريع

### المتطلبات
- Node.js >= 18.0.0
- pnpm >= 8.0.0

### التثبيت والتشغيل
```bash
# تثبيت الاعتماديات
pnpm install

# تشغيل بيئة التطوير
pnpm dev

# البناء للإنتاج
pnpm build

# معاينة البناء
pnpm preview
```

### الفحوصات
```bash
pnpm typecheck   # فحص الأنواع
pnpm lint        # فحص الكود
pnpm format      # تنسيق الكود
pnpm test        # تشغيل الاختبارات
pnpm validate    # تشغيل جميع الفحوصات
```

## 🌐 النشر

### Vercel (مستضاف حالياً)
```bash
# النشر المباشر
vercel --prod
```

### GitHub Actions (CI/CD)
- **Validate & Lint**: فحص الكود والأنواع
- **Unit Tests**: اختبارات الوحدة
- **Security Audit**: فحص أمني
- **Build**: بناء الإنتاج
- **Deploy**: نشر تلقائي إلى Vercel

## 📁 هيكل المشروع

```
src/
├── app/           # تطبيق رئيسي (صفحات، مكونات)
├── content/       # محتوى CMS
├── features/      # ميزات (auth, core, ...)
├── sanity/        # تكامل Sanity CMS
├── shared/        # خدمات مشتركة
├── styles/        # أنماط CSS
└── utils/         # أدوات مساعدة
```
