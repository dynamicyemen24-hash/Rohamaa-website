# مستعرض الوسائط المحترف - Professional Media Viewer

## نظرة عامة
مستعرض الوسائط المتكامل لموقع "رحماء بينهم" - يدعم الصور والفيديوهات مع واجهة احترافية.

## الميزات الرئيسية

### 1. واجهة العرض المتكاملة
- **إطار مدمج في الصفحة** - لا يُفتح نافذة ملء الشاشة
- **قابل لتغيير الحجم بالسحب** - ResizableFrame component
- **إخفاء/إظهار القائمة الجانبية** - Sidebar toggle
- **انتقالات سلسة** - باستخدام Framer Motion

### 2. القائمة الجانبية الذكية
- **عرض جميع الوسائط** - صور وفيديوهات
- **البحث الفوري** - بدون إعادة تحميل
- **التصفية حسب النوع** - images / videos / documents
- **عرض مميزات الوسائط** - العنوان، النوع، المدة، التاريخ
- **تحديد العنصر الحالي** - مؤشر بصري واضح

### 3. أدوات التحكم

#### للصور:
- تكبير / تصغير 🔍
- Fit Width / Fit Height 📐
- التحجيم الأصلي
- التدوير 🔄
- القلب الأفقي والعمودي
- التنزيل 💾
- المشاركة 🔗
- عرض المعلومات ℹ️

#### للفيديو:
- تشغيل / إيقاف ⏯️
- شريط التقدم
- الوقت الحالي والإجمالي
- التحكم بالصوت 🔊
- سرعة التشغيل (0.5x - 2x)
- Picture in Picture
- ملء الإطار

### 4. وسائل التنقل
- **الأسهم ← →** - من لوحة المفاتيح
- **عجلة الماوس** - للتنقل بين الوسائط
- **التمرير اللمسي (Swipe)** - على الهواتف
- **نقر المصغرات** - الانتقال لأي عنصر
- **الانتقال الدائري** - عند الوصول لنهاية القائمة

### 5. التكامل مع لوحة التحكم
- **جلب البيانات من Sanity API**
- **تحديث تلقائي** عند تعديل البيانات
- **دعم الحالات** (منشور/مسودة/مؤرشف)
- **دعم الوسوم والألبومات**

### 6. تحسينات الأداء
- **Lazy Loading** - تحميل الصور عند الحاجة
- **الكاش الذكي** - لتقليل طلبات API
- **تحميل الفيديو عند الطلب** - onDemand video loading

### 7. تجربة المستخدم
- **الوضع الليلي والنهاري** - كامل
- **تصميم متجاوب** - Responsive
- **إمكانية الوصول** - Accessibility support
- **دعم لوحة المفاتيح** - Keyboard navigation

## الاستخدام

```tsx
import { ProfessionalMediaViewer } from '@/app/components/MediaViewer';

// في صفحة الوسائط
<ProfessionalMediaViewer
  options={{
    autoPlay: false,
    showSidebar: true,
    sidebarViewMode: 'expanded',
    theme: 'charity',
    enableKeyboardNav: true,
    enableMouseWheel: true,
    enableSwipe: true,
  }}
/>
```
